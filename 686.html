Damit Sie Updates von dieser Website beziehen können, muss Skripting aktiviert sein.
Wenn Sie über diese Site Updates suchen und herunterladen, müssen Sie Ihre Sicherheitseinstellungen so ändern, dass ActiveX-Steuerelemente und Active Scripting aktiviert sind. Oder richten Sie diese Site als vertrauenswürdige Site ein. Mit Ihren Sicherheitseinstellungen werden dann weiterhin unsichere ActiveX-Steuerelemente und Scripting auf anderen Sites blockiert, Sie können jedoch Updates herunterladen:
Klicken Sie in Internet Explorer auf Extras und dann auf Internetoptionen. Klicken Sie auf der Registerkarte Sicherheit auf das Symbol Vertrauenswürdige Sites. Klicken Sie auf Sites, und fügen Sie dann diese Websiteadressen nacheinander der Liste hinzu: Sie können nur eine Adresse auf einmal einfügen und müssen nach jeder Adresse auf Hinzufügen klicken:
http://*.update.microsoft.com https://*.update.microsoft.com http://download.windowsupdate.com
Hinweis: Möglicherweise müssen Sie die Option Für Sites dieser Zone ist eine Serverüberprüfung (https:) erforderlich deaktivieren, um alle Adressen einzugeben.
